### 🗺️ ЕГКН Фильтр — Chrome-расширение для кадастра РК
`2025` · [GitHub](https://github.com/Narimannmn/egkn-filters)

Расширение для удобной фильтрации земельных участков на портале [map.gov4c.kz/egkn](https://map.gov4c.kz/egkn/) — без бесконечного кликанья по карте. Сделал для себя, пока искал землю под ЛПХ, и опубликовал.

- Штатный сайт не имеет сводного списка — только карта. Расширение добавляет **панель со списком, фильтрами и пагинацией**
- **Полная загрузка всех страниц** выбранного слоя с прогресс-баром (`загружено 210 из 402`)
- **Двухуровневая фильтрация** по статусу, виду права и другим полям с подсчётом значений
- **Стриминговая загрузка** волнами по 20 параллельных запросов через TanStack Query
- **Shadow DOM** — полная изоляция стилей расширения от хост-сайта
- Кэш на всю сессию, копирование ID одним кликом, авто-подстановка района из сессии
- Работает в Chrome / Edge / Brave / Opera / Yandex Browser

`React 19` `TypeScript` `TanStack Query v5` `Radix UI` `Shadow DOM` `Vite` `Manifest V3`
